## Installation
Add this line to your application's Gemfile:

```ruby
gem 'sukashi', git: 'https://github.com/kmdtmyk/sukashi', ref: '<commit_hash>'
```

And then execute:
```bash
$ bundle install
$ rails g sukashi:install
```
